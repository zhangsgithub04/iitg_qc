Basic Quantum Gates

Quantum gates are operations that change qubit states. This page teaches the main gates using matrices, Dirac notation, circuit symbols, and Bloch-sphere intuition.

1. A qubit before gates

A single qubit is written as a superposition of the basis states |0> and |1>.

State vector form

|psi> = alpha|0> + beta|1> = [ alpha ]
　　　　　　　　　　　　　[ beta ]

The complex numbers alpha and beta are amplitudes. Their squared magnitudes give measurement probabilities:

P(0) = |alpha|^2,   P(1) = |beta|^2,   |alpha|^2 + |beta|^2 = 1

What a gate does

A gate is usually represented by a matrix. Applying a gate means multiplying the matrix by the qubit state vector.

new state = gate x old state

Gates do not choose a measurement result. They rotate, flip, or phase-shift the state before measurement happens.

2. The X gate: quantum NOT

X Meaning

The X gate swaps |0> and |1>. It is the quantum version of a NOT gate.

X|0> = |1>
X|1> = |0>

Matrix

X = [ 0  1 ]
　　[ 1  0 ]

On the Bloch sphere, X is a 180 degree rotation around the x-axis.

3. The Z gate: phase flip

Z Meaning

The Z gate keeps |0> the same but changes the sign of |1>. It does not change measurement probabilities in the computational basis by itself.

Z|0> = |0>
Z|1> = -|1>

Why phase matters

The states |+> = (|0> + |1>)/sqrt(2) and |-> = (|0> - |1>)/sqrt(2) both give 50/50 measurement in the |0>, |1> basis, but they behave differently under later gates.

Z|+> = |->

4. The H gate: Hadamard

H Creates and removes superposition

The Hadamard gate maps basis states into equal superpositions.

H|0> = |+> = (|0> + |1>)/sqrt(2)
H|1> = |-> = (|0> - |1>)/sqrt(2)

Matrix

H = 1/sqrt(2) [ 1   1 ]
　　　　　　 [ 1  -1 ]

Applying H twice returns the original state:

H(H|0>) = |0>,   H(H|1>) = |1>
Teaching tip: H is often introduced as "make a qubit random," but that is incomplete. H creates a controlled superposition with phase, not classical randomness.

5. S and T gates: smaller phase rotations

S Quarter-turn phase gate

The S gate multiplies the |1> amplitude by i.

S = [ 1  0 ]
　　[ 0  i ]
S|1> = i|1>

T Eighth-turn phase gate

The T gate gives an even smaller phase shift.

T = [ 1    0 ]
　　[ 0  exp(i*pi/4) ]

S and T are important because quantum algorithms depend on controlling phase, not just flipping 0s and 1s.

6. CNOT: a two-qubit gate

CNOT has two inputs: a control qubit and a target qubit. It flips the target only when the control is |1>.

Truth table on basis states

InputOutputMeaning
|00>|00>Control is 0, target unchanged.
|01>|01>Control is 0, target unchanged.
|10>|11>Control is 1, target flips.
|11>|10>Control is 1, target flips.

Circuit symbol

control
target 

The vertical connection means the target operation depends on the control qubit.

CNOT can create entanglement. Example: start with |00>, apply H to the first qubit, then CNOT. The result is (|00> + |11>)/sqrt(2), a Bell state.

9. Suggested lesson flow

Step 1

Review |0>, |1>, amplitudes, and measurement probability.

Step 2

Teach X first because it behaves most like a familiar classical NOT gate.

Step 3

Teach H and Z together to show why phase matters even when probabilities look unchanged.

Step 4

Finish with CNOT and the Bell-state recipe: |00> -> H on first qubit -> (|00> + |10>)/sqrt(2) -> CNOT -> (|00> + |11>)/sqrt(2). This gives students their first concrete example of entanglement.
